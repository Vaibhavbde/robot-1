⚡ Neon Sorting Visualizer

Experience sorting algorithms like never before! This neon-themed, animated sorting visualizer is built with Java Swing, bringing 6 classic algorithms to life with glowing bars, smooth animations, and real-time stats.

🎬 Features

6 Classic Sorting Algorithms — Bubble, Selection, Insertion, Merge, Quick, Shell

Neon Dark Theme — gradient bars with radiant glow effects

Live Stats — watch comparisons and swaps update in real time

Speed Control — adjust from slow-motion to lightning-fast

Pause / Resume — freeze mid-sort to inspect every step

Ascending / Descending Toggle — switch the sorting order instantly

Sweep Animation — enjoy a satisfying green wave when sorting finishes

🚀 How to Run

Run:

Windows: run.bat

Linux/macOS: run.sh

🎨 Color Guide
Color	Meaning
🔵 Cyan	Unsorted bar
🔴 Pink	Currently being compared or swapped
🟡 Yellow	Pivot element (Quick Sort)
🟢 Green	Sorted / finalized bar
📊 Algorithm Complexity
Algorithm	Best	Average	Worst	Space
Bubble Sort	O(n)	O(n²)	O(n²)	O(1)
Selection Sort	O(n²)	O(n²)	O(n²)	O(1)
Insertion Sort	O(n)	O(n²)	O(n²)	O(1)
Merge Sort	O(n log n)	O(n log n)	O(n log n)	O(n)
Quick Sort	O(n log n)	O(n log n)	O(n²)	O(log n)
Shell Sort	O(n log n)	O(n log² n)	O(n²)	O(1)
